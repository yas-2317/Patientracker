# うつ病治療ダッシュボード（Depression Treatment Dashboard）

うつ病患者の治療経過を可視化・分析するための臨床意思決定支援ダッシュボードです。

## 概要

患者の評価スコア推移・治療エピソード比較・予後予測をワンページで確認できます。

## 技術スタック

- **Next.js 14**（App Router）
- **TypeScript**
- **Tailwind CSS**
- **Recharts**（グラフライブラリ）

## 画面構成

### Tab 1 - Patient Timeline
- PHQ-9 / QIDS / HAM-D / MADRS 切替表示
- 合計スコアの時系列折れ線グラフ（治療イベント縦線付き）
- PHQ-9項目別ヒートマップ（9項目 × 測定日）
- 期間フィルタ（直近3か月 / 6か月 / 全期間）
- 患者サマリ・現在治療・最新スコアのサイドバー

### Tab 2 - Episode Comparison
- Before / During / After の3フェーズ比較（棒グラフ）
- PHQ-9項目別レーダーチャート（Before vs After）
- 反応率・寛解率・平均改善時間のKPI表示

### Tab 3 - Prognosis & Prediction
- ペルソナ分析（不眠優位型・不安併存型・社会機能低下型）
- 治療シナリオ別アウトカム予測（エリアチャート）
- 類似患者テーブル（5名分）
- ⚠️ 免責注記：本ツールは参考情報であり、診断・治療の決定は医師が行ってください

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 注意事項

- 本ダッシュボードはすべてダミーデータを使用しており、実際の患者情報は含まれていません
- PC幅（1280px以上）での使用を推奨します
- 本システムの予測は統計的推定であり、医療的判断を代替するものではありません
